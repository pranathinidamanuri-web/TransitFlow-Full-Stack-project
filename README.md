\# TransitFlow



TransitFlow is a Bus Pass \& Ticket Reservation System built using ReactJS, Node.js, Express, and SQLite. It allows passengers to view routes, book tickets, cancel bookings, and manage travel efficiently.



\---



\## 🚀 Features



\- View available bus routes and schedules  

\- Book bus tickets  

\- Prevent double seat booking  

\- Cancel booked tickets  

\- View booking history  

\- Data stored using SQLite database  



\---



\## 🛠 Tech Stack



\- Frontend: ReactJS  

\- Backend: Node.js + Express  

\- Database: SQLite
